# Accessories By Melek

A premium luxury accessories e-commerce website built with HTML, CSS, and JavaScript.

## Features

✨ **Modern Design**
- Dark luxury theme with gold accents
- Fully responsive design (mobile, tablet, desktop)
- Smooth scrolling and animations
- Premium gradient effects

📦 **Product Showcase**
- Beautiful product grid with hover effects
- Product cards with images and pricing
- Easy product browsing

🤝 **Social Integration**
- Instagram profile link
- WhatsApp group invite
- WhatsApp floating chat button
- Direct messaging support

📧 **Contact Form**
- Custom inquiry form
- User-friendly interface
- Form validation

## Getting Started

### Option 1: Live on GitHub Pages
1. Go to **Settings** → **Pages**
2. Select **Deploy from a branch**
3. Choose **main** branch and **/ (root)** folder
4. Click Save
5. Your site will be live at: `https://melekaura400-crypto.github.io/Accessoriesbymelek/`

### Option 2: Local Development
Simply open `index.html` in your web browser.

## Important Setup Notes

⚠️ **Before Publishing, Update These Links:**

1. **WhatsApp Group Link** (Line 217)
   - Replace `https://chat.whatsapp.com/your-group-link` with your actual group invitation link
   - Get it from: WhatsApp Group → Menu → Invite via link

2. **WhatsApp Direct Chat** (Line 235)
   - Replace `https://wa.me/your-phone-number` with your phone number
   - Format: `https://wa.me/923001234567` (include country code, no + symbol)

3. **Instagram Profile** (Line 152)
   - Replace `https://instagram.com/accessoriesbymelek` with your actual Instagram handle

## Customization

### Colors
- **Gold/Primary**: `#d4af37`
- **Dark Background**: `#121212`
- **WhatsApp Green**: `#25d366`
- **Instagram Pink**: `#e1306c`

Edit these in the `<style>` section to match your brand.

### Products
Edit the product cards in the **Products Section**:
- Update product names, prices, and images
- Add more cards by duplicating the `product-card` div

### Content
All text content is in the HTML sections:
- Hero section
- About us
- Product descriptions
